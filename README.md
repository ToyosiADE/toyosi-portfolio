# Data Cleaning in World Layoffs Dataset

This project focuses on cleaning and transforming the World Layoffs dataset using SQL.  
The goal was to prepare the data for accurate analysis and visualization.

## Key Steps
- Removed duplicate records using ROW_NUMBER().
- Standardized text fields (industry, country, etc.) for consistency.
- Converted text-based dates to proper DATE format.
- Handled null and irrelevant values to ensure data quality.

## Tools Used
- SQL Server / MySQL
- Excel / Power BI (for visualization)
